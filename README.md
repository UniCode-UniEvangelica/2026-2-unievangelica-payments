# 2026-2-unievangelica-payments

Repositório didático para prática de testes automatizados com Pytest.

Disciplina: Teste de Software
Período: 7º

---

## Objetivo

Executar testes automatizados, identificar falhas e corrigir defeitos no código.

---

## Execução

1. Criar ambiente virtual (opcional)

```bash
python -m venv .venv
```

2. Ativar ambiente

Mac/Linux:
```bash
source .venv/bin/activate
```

Windows:
```bash
.venv\Scripts\activate
```

3. Instalar dependências

```bash
pip install -r requirements.txt
```

4. Executar testes

```bash
pytest
```

---

## Atividade em sala

* Identifique os testes que falham
* Analise o motivo técnico
* Corrija as funções
* Execute novamente até todos passarem
* Realize commit com mensagem clara
